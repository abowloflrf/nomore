# Ghost 主题 nomore

一个简洁的同时又美观耐看不普通的 Ghost 主题，适合开发者个人博客

## Featrue List

*   移动端优化
*   中文显示友好
*   代码高亮（使用 highligh.js 若需要支持更多冷门语言，请 clone 后修改配置重新 build）
*   支持的 Ghost 页面：index,post,tag,author,page
*   文章页面图片支持点击缩放（使用 zoom.js）
